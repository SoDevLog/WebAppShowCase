WebApp Dynamic ShowCase

This application is a web site an ASP.NET Application that dynamicly present content into pages that you can organise in a user's menu. This project is managed by Mabyre who is a member of the company SoDevLog

Environment

  • Visual Studio 2013 +
  • ASP.NET 4.5 +

Get started

  1. Clone repository
  2. Open solution in Visual Studio 2013 or more
  3. Builld and run solution to load website in the browser
  4. Use admin/admin to log into website administration
